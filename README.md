# Backbone Adventure 

The application is simple Todo List over Backbone, 
everything should be nice and working (but with no validations whatsoever).


### Web Application Description:


#####Server Side:
 * Web Server: WEBrick
 * Framework: Rails 3.2.13
 * DB: SQLite
 * Templating: Erb, Rabl
 * CSS: Boostrap
     
#####Client Side:
 * JS: Bootstrap.js
